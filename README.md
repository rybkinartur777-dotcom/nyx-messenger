# Nyx — Анонимный защищённый мессенджер

Кроссплатформенный мессенджер с фокусом на конфиденциальность и анонимность.

## 🚀 Быстрый старт

### Требования
- Node.js 18+
- npm или yarn

### Установка

```bash
# Клонировать репозиторий
cd nyx

# Установить зависимости клиента
cd client
npm install

# Установить зависимости сервера
cd ../server
npm install

# Создать папку для базы данных
mkdir data
```

### Запуск

```bash
# Терминал 1: Запустить сервер
cd server
npm run dev

# Терминал 2: Запустить клиент
cd client
npm run dev
```

Откройте http://localhost:3000 в браузере.

## 🔐 Безопасность

- **E2E шифрование**: X25519 + XChaCha20-Poly1305 (libsodium)
- **Анонимная регистрация**: Без телефона и email
- **Уникальный ID**: Автоматически генерируется (NYX-XXXXXXXX)
- **Хранение**: Сообщения хранятся только в зашифрованном виде
